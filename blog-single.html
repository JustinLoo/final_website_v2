Advanced Database App PHP | PDO | AJAX | SQL

Published on August 2021, project finished on June. By Justin Loo

The Register Screen on my app. Once you have an account, you will be able to Log-In

This app was made in Visual Studio Code. It uses MySQL and Apache Server on XAMPP Server Panel. Languages I used: HTML, CSS, JavaScript, PHP, SQL

The first step to creating this app was making the servers. This was pretty simple. Download XAMPP with all the servers are already installed for you; create the required databases,rows, and colums in phpmyadmin. Second, you must create an outline on what you will include on your app. Next, create all HTML and CSS and create a simple black form with Bootstrap. In this case, my app included a login system, student system with CRUD, and AJAX. This project took 4-5 months and there was a lot to learn. With This project, I extended my knowledge of the databases and mainly learned backe-end with php. I plan on creating more back-end applications in the future!

This app will be live soon! For now it is on my Github, until I can get a hosting service. Github Pages does not support PHP and advanced websites.

Beginner Database App

Published on August 2021, project finished on December 2020. By Justin Loo

Full View of all the apps and programs I used. This app is ran in the Python Shell

This app was made in Visual Studio Code. It uses MySQL and Apache Server on XAMPP Server Panel. Languages I used: Python, SQL

The first step to creating this app was making the servers. This was pretty simple. Download XAMPP with all the servers are already installed for you; create the required databases,rows, and columns in phpmyadmin. Second, make a new python script and connect to the database using SQL Querys. In this project, I made a simple password system(not regitser or anything fancy) that has the idea of a teacher adding students, editing, deleteing values.(CRUD)

This app will be is a Console App. If you want to run it download files on GIthub and run it throught the most recent Python Shell.
